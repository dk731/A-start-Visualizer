# A\* algorithm visualizer

## Basic information

A\* algorithm visualizer using python and pygame (for UI).

## Core functionality:

- 1st LMB click - place start point
- 2nd LMB click - place finish point
- RMB hold and drag - place walls
- ENTER - start searching solution
- ESC - reset
